# CustomRanks

A resource pack for Minecraft Java and Bedrock that adds custom ranks like Better Ranks for ItemsAdder. This pack provides a free and easy-to-use alternative with no plugin requirements - just install the resource pack!

## Features

- **Custom Ranks**: Extensive collection of pre-made rank icons
- **Cross-Platform**: Works with both Minecraft Java and Bedrock Edition
- **Plugin-Free**: No additional plugins required
- **Editable Assets**: All PNG files can be easily customized when extracted
- **Free & Open**: Completely free to use and modify

## Installation

1. Download the latest release
2. Place the resource pack in your Minecraft resource packs folder
3. Activate the resource pack in your Minecraft settings
4. Enjoy your custom ranks!

## Usage

To use these ranks in chat or as prefixes/suffixes, copy the corresponding symbols into your chat formatting or permissions plugin configuration.
